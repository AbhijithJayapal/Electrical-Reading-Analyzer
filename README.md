# Electrical Reading Analyzer

A beginner Python project that analyzes electrical voltage readings entered by the user.

## Features
- Reads multiple voltage readings
- Classifies readings as:
  - Undervoltage
  - Normal
  - Overvoltage
- Handles invalid inputs using exception handling
- Displays the analyzed results

## Skills Used
- Python
- Functions
- Loops
- Conditional Statements
- Lists
- Error Handling
